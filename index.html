1.Clash of Clans MiniGame

2.Pong Game

This game is the replica of the ping pong (TT) in code.You have to move the player to bounce the ball towards the opponent who is a computer

You have 5 lives, once all 5 are gone the game is over

Can you beat the unbeatable robot and win the game

3.Space Shooter Game

This game is based on the shpace mobs game on the app/play store.

4.Trex Runner Game

This game is the representation of the non internet Trex runner game on google chrome

You are a runnaway dinosaur who is being chased by hunters.

You have to dodge cactuses,birds etc to run faster than the hunters who are close behind

Will you be able to run faster than your pursuers or will the hunters prevail and Hunt You Down.

This game is inspired by the Carrom Board game in india,
